# **App Name**: Baalvion

## Core Features:

- Unified Business Dashboard: A central overview for all managed businesses, displaying key performance indicators like revenue, profit, employee count, and domains in a concise, role-specific view.
- Detailed Business Management: Pages for administrators and co-founders to view and manage specific details, financial metrics (including revenue, profit, equity splits, and currency conversion using mock FX rates) for individual businesses.
- Employee & Role Management: Interface for administrators to manage employees and assign predefined roles (Admin, Investor, Co-Founder, Employee), controlling access and views across the system.
- Dynamic Mock Data Presentation: Robust UI components for tables, charts, and cards that dynamically populate with real-looking mock data (e.g., company names, realistic percentages, varying currency values), ensuring no empty states.
- Multi-Tenant Business Group Selector: A user interface mechanism (e.g., dropdown in the navigation) to switch between different 'client' business groups, demonstrating the multi-tenancy architecture with mock data.
- AI-Powered Performance Insights: An AI tool that generates a concise, context-aware summary of key business performance trends, potential anomalies, or noteworthy events based on the available mock data, aiding quick decision-making.

## Style Guidelines:

- Primary interactive color: A confident, deep business blue (#123BCC) for active elements, buttons, and crucial calls to action, evoking reliability and professionalism.
- Main background color: A clean, very light off-white with a subtle blue tint (#F9FAFC) for the main content area, providing a spacious and airy feel.
- Accent color: A clear sky blue (#44C0ED) to highlight secondary information, notifications, or provide a refreshing visual contrast to the primary and background colors.
- Sidebar navigation background: A professional, dark charcoal gray (#232428) providing a stark contrast for the fixed dark sidebar navigation, ensuring high readability for white text.
- All text: 'Inter', a modern grotesque sans-serif, chosen for its exceptional legibility, objective appearance, and versatility across headlines and body text, supporting a clean, production-grade interface.
- Icons: Clean, professional line icons (compatible with Lucide Icons used in shadcn/ui) that clearly represent business, finance, user management, and operational functions, maintaining visual consistency.
- Layout: A classic two-column dashboard layout featuring a dark, fixed-position left sidebar navigation and a wide, clean white/light main content area optimized for data-rich displays.
- Animations: Subtle, purposeful transitions and hover effects to provide feedback on user interactions and smooth data loading or state changes without being distracting.
